++ Date Type

    - To store date values in memory Javascript provides "Date" class.
    - You have to use Date class constructor to create a date object and store date in memory.

        class Date{
            constructor(){}
        }
        var mfd = new Date(); // It loads current date and time

        var mfd = new Date("yy-mm-dd");
    ++ Date and Time Methods
    - getHours()        : It returns hours 0 to 23
    - getMinutes()      : 0 to 59
    - getSeconds()      : 0 to 59
    - getMilliSeconds() : 0 to 99
    - getDate()         : returns date number [0 - 31]
    - getDay()          : returns weekday number [0 = Sunday]
    - getMonth()        : returns month number [0 = Jan]
    - getFullYear()     : returns full year number [2024]
    - getYear()         : It is obsolete, no long in use. Because of the Y2K problem. (returns 124 for 2024)
    - toLocaleDateString: returns in the format of the machine executed upon
    - toLocaleTimeString: returns in the format of local machine


    - You can dynamically set a new date or time by using following methods;
        setHours()
        setMinutes()
        setSeconds()
        setMilliSeconds()
        setDate()
        setMonth()
        setYear()
