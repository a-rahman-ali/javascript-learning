++ Configuration of Array
    - Array configuration comprises of 2 phases
        a) Declaration
        b) Initialization
    - Declaration is same as general variables.
    - Initialization or Assignment of memory is mandatory.
    - Memory is assigned or initialized for array by using 2 methods
        a) []      - meta character
        b) Array() -- constructor

        var products; // Declaration
        products = []; // Assignment
        products = new Array();
    - [] allows range of values | allows multiple values
    - new Array()
        new     => Dynamic Memory Allocating Operator
        Array() => Creates an array in memory
        ()      => constructor
        (size)  => You can define size, but size is not constant.

    - Syntax:
        var products = [];
        var categories = new Array();

    ++ Storing values in array:
    - Values are stored in array with reference of its properties.
    - Array Property maps to its index number. (refer to example-1 18_arrays.html)
            Property "0"  =>  Index 0
            values[0]
            values["0"]
    - You can store any type of values in array. It can be primitive or non-primitive. (refer to example-2 18_arrays.html)
